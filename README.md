# 📊 Financial Modelling

## Overview

This repository showcases financial modelling projects completed during my MSc in Accounting and Finance at the University of Leeds.

The projects demonstrate practical valuation techniques used in investment banking, corporate finance and equity research.

---

## Topics Covered

- Discounted Cash Flow (DCF)
- Weighted Average Cost of Capital (WACC)
- Capital Asset Pricing Model (CAPM)
- Free Cash Flow to Firm (FCFF)
- Enterprise Value
- Equity Value
- Sensitivity Analysis
